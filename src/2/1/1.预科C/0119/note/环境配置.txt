7.linux简单指令的使用
    (1)命令行终端提示符 china@ubuntu:~$
        china 用户名
        @     分隔符,用来分隔用户名和主机名
        ubuntu 主机名
        :      分隔符 用来分隔主机和当前工作目录
        ~      当前所在的工作路径 家目录
        $      表示当前用户是普通用户
    
    (2)特殊字符
        ~       家目录
        $       普通用户
        #       超级管理员
        .       表示当前文件夹
        ..      当前文件夹的上一级目录
        /       根目录,根目录是linux中最高级的目录
        /home   不是家目录,所有用户的家目录存在的地方
                eg:
                    /home/china

    (3)目录和文件夹的区别
        文件夹他是一个包含和包含的关系,可以通过文件夹看到所有文件的大小
        目录只是一个目录的索引
    
    (4)linux的简单指令
        注意:linux当中的指令的运行格式
            指令 参数

        cd: change directory
            作用就是切换你的目录
            例子:
                进入共享文件夹
                cd /mnt/hgfs/你对应的共享文件夹
            
            小技巧:
                再敲指令的时候可以使用tab键进行补全

        ls:
            list 列表 清单 
                列出你当前目录下的所有文件绿色-

        gcc:
            编译器,他是linux下最强大的编译器
            我们以后写好代码就是通过这个编译器来进行一个编译的
            编译的语法:
                1)
                gcc 要编译的文件名
                会生成一个a.out的可执行文件
                在当前目录下找到这个可执行文件就OK了
                ./:当前目录的意思
                ./a.out

                2)
                gcc 要编译的文件名 -o 你需要指定生成的文件名
                执行:./指定生成的文件名
        
        history:可以查看你之前敲过的指令
                最多查看2000条
                使用方向键上下就OK了

        pwd:print work directory
            打印当前的一个工作目录
            语法:
                直接输入pwd就行了

        练习:
            写一个从键盘输入两个数,输入他们的和
            在linux上编译运行出来
                代码规范
                语法错误
                ...

8.gcc编译器
    linux中用来编译C代码的工具

    程序运行的流程
        (1)编辑代码 
            文件的格式是 .c文件,存放在共享文件夹中(建议大家按日期来存放)
            注意事项
                1)除了你的注释和""中,代码不要出现任何的中文字符
                2)所有的成对符号一次性先敲好,以防万一漏了一半
                    "" () {} [].....
                3)缩进问题
                    为了让这个代码更加有层次感
                    缩进就是通过TAB键来实现

        (2)编译代码
            在Ubuntu的共享文件夹下面进行编译
            编译:把人看得懂的代码翻译成机器看得懂的代码
            编译的指令上面有:
                1)
                gcc 要编译的文件名
                会生成一个a.out的可执行文件
                在当前目录下找到这个可执行文件就OK了
                ./:当前目录的意思
                ./a.out

                2)
                gcc 要编译的文件名 -o 你需要指定生成的文件名
                执行:./指定生成的文件名

            编译有一个严重的"BUG"
                它只能够找到语法的错误,不能找到逻辑的错误
                假设你出现了逻辑上的错误:
                    解决方法:重新对你的程序进行构思,重新调试

        (3)执行代码
            ./可执行文件名 
            需要进入可执行程序所在的文件夹下来执行这个可执行程序


9.linux下最强大的一个编辑器 vi/vim
    一款不具备操作界面 但是功能齐全的编辑器
    用法:
        使用vi创建文件
        vi/vim 文件名
        文件存在就是打开,如果不存在就自动创建
    
    命令模式:
        1)a/i进入编辑模式
        i:从光标的当前位置开始输入
        a:从光标的下一个位置开始输入

        2)快捷键(需要在命令模式中进行输入)
        yy        复制一行
        nyy       复制n行
        dd        剪切一行
        ndd       剪切N行
        p         粘贴
        u         撤销
        ctrl+r    反撤销
        gg        回到文件开头

        3)文件写完了的退出操作
            首先按下ESC进入命令模式
            输入一个 冒号 :  
            w       保存文件
            q       退出文件 
            q!      强制退出文件
            wq      保存并且退出文件

10.关于如何在linux创建目录和文件
    touch:
            创建普通文件(文本文件 图片文件 压缩文件...)
        用法:
            touch 文件名(可以带路径的)
        eg:
            touch 1.txt  :在当前的路径下创建一个1.txt的文本文件
            假设我们现在在note目录下,要去code目录下创建一个3.c文件 
            touch 路径/3.c 
                路径:../code/3.C

    mkdir:
            创建一个目录
        用法:
            mkdir 目录名(可以带一个路径)
        
        eg:
            mkdir ddd

        练习:
            进入家目录: cd ~
            创建一个目录 abc,在不进入abc前提下新建一个文件1.txt和一个目录bcd在abc下

    rm:
        remove

        删除一个普通文件:
            rm 你要删除的文件名(带路径)

        删除一个目录的时候:
            rm -rf 你要删除的目录名(带路径)
